# Resume Screening System

## Overview
This project uses Machine Learning and NLP techniques to automatically screen resumes based on job descriptions.

## Tech Stack
- Python
- NLP (TF-IDF)
- Machine Learning (Naive Bayes / Logistic Regression)
- Flask / Streamlit

## Features
- Resume classification
- Job matching system
- PDF resume parsing
- Automated screening process

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py
