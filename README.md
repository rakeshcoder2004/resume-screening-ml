# Resume Screening System

## 📌 Overview
This project is an AI-based Resume Screening System that automates the process of evaluating resumes based on job descriptions. It uses Natural Language Processing (NLP) techniques and Machine Learning models to classify resumes into relevant job roles, reducing manual effort and improving hiring efficiency.

---

## 🧠 Problem Statement
Manual resume screening is time-consuming and inefficient. Recruiters often spend hours filtering candidates. This system aims to:
- Automate resume classification
- Improve candidate-job matching
- Reduce human effort in recruitment

---

## ⚙️ Tech Stack
- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **NLP Technique:** TF-IDF Vectorization  
- **ML Models:** Naive Bayes / Logistic Regression  
- **Framework:** Streamlit / Flask  
- **Other Tools:** PDF parsing (PyPDF / similar)

---

## 🔍 How It Works
1. Resume is uploaded (PDF/Text)
2. Text is extracted and cleaned (preprocessing)
3. TF-IDF converts text into numerical features
4. Machine Learning model predicts the job role
5. Output is displayed to the user

---

## ✨ Features
- 📄 Resume upload and parsing  
- 🤖 Automated job role prediction  
- ⚡ Fast and efficient screening  
- 📊 NLP-based text processing  
- 🌐 Simple user interface
